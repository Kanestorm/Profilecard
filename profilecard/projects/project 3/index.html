Password Generator

Project Overview

The Password Generator is a web-based application that creates secure and random passwords based on user inputs. It allows users to enter their first name, last name, and mobile number and generates a unique password using a mix of characters from these inputs, along with special symbols. This tool helps users create personalized yet strong passwords for better security.

Features

  • User Input-Based Password Generation: Uses first name, last name, and mobile number to create a password.
  • Special Characters Inclusion: Adds special symbols ($#%@&) to enhance password strength.
  • Randomization: Selects characters randomly from user inputs and special symbols.
  • User-Friendly Interface: Simple design with an easy-to-use layout.
  • Responsive Design: Works on both desktop and mobile devices.

Technologies Used

  • HTML: Provides the structure of the webpage.
  • CSS: Styles the interface for a clean and user-friendly experience.
  • JavaScript: Implements password generation logic and handles user interactions.

Summary

The Password Generator is a web-based tool that creates secure passwords using a mix of user inputs (first name, last name, and mobile number) along with special characters. It features a simple interface, random character selection, and a responsive design. Built with HTML, CSS, and JavaScript, it helps users generate personalized yet strong passwords. Future improvements could include better randomization, password strength indicators, copy-to-clipboard functionality, and customizable password length. This project is great for beginners learning JavaScript logic and UI development.
