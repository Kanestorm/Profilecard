Number Guessing Game

Project Overview

The Number Guessing Game is an interactive web-based game where users try to guess a randomly generated number between 0 and 9. The game provides hints to guide players by indicating whether they should guess higher or lower. It continues until the correct number is guessed, displaying a success message when the player wins.

Features

  • Random Number Generation: The game selects a random number between 0 and 9.
  • User Input: Players enter a number to make a guess.
  • Real-Time Feedback:
    • Alerts notify players whether to guess a higher or lower number.
    • Displays "You won the game" when the correct number is guessed.
  • Interactive UI:
    • Beautifully designed CSS-styled interface with a color theme.
    • A hover effect on the submit button for better user interaction.
  • Responsive Layout: The game adjusts well across different screen sizes.

Technologies Used

  • HTML: Provides the structure of the game.
  • CSS: Enhances the visual design with styles and effects.
  • JavaScript: Implements game logic, random number generation, and user interaction.

Summary

The Number Guessing Game is a simple web-based game where users guess a randomly generated number between 0 and 9. It provides real-time feedback, guiding players to guess higher or lower until they win. The game features a styled UI with responsive design, built using HTML, CSS, and JavaScript. Future improvements could include difficulty levels, attempt tracking, a restart button, and enhanced visuals. This project helps beginners practice JavaScript logic, event handling, and UI design.
